CSS经典三栏布局方案

发表于 2017-05-03 | 分类于 css

一道经典面试题，假如高度已知，请写出三栏布局，其中左右宽度个300，高度已知

首先初始化一下样式

浮动布局

浮动布局兼容性很好，是目前使用较多的布局方式，使用时需要注意清楚浮动

绝对定位方式

绝对定位方式是根据定位属性去直接设置元素位置，会脱离文档流，影响后面的dom元素排列

flex方式

flexIE9以下不兼容 https://caniuse.com/#feat=flexbox

grid 方式

grid布局很强大，但是兼容性较差 https://caniuse.com/#feat=css-grid

table布局

table布局使用起来方便，兼容性也不存在问题，但是对seo不够友好
