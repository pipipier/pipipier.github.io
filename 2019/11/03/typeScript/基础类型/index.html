基础类型

发表于 2019-11-03 | 分类于 typeScript

布尔值

使用 boolean 定义布尔值类型。

数字

使用 number 定义数值类型。
和JavaScript一样，TypeScript里的所有数字都是浮点数，支持十进制、十六进制、二进制和八进制。

字符串

使用 string 定义字符串类型。
和JavaScript一样，可以使用双引号（ “）或单引号（’）表示字符串。
还可以使用模版字符串，${expr} 用来定义多行文本和内嵌表达式。
